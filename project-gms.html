RottIng - LinkedIn Post Generator

RottIng is like a supercharged app for the LinkedIn crowd. It takes those cringy, overly serious LinkedIn vibes and cranks them up to absurdity, generating wild, meme-worthy, over-the-top posts that are pure chaos â€” think Ohio-core, giga-skibidi, and full-mogged energy.

The project was developed during a hackathon where our team wanted to create something that could both entertain and comment on the often exaggerated nature of professional social media. Using advanced AI techniques, RottIng analyzes patterns from LinkedIn posts and generates content that mimics and amplifies their style to comedic effect.

Key Features

  • AI-powered text generation specifically tuned for LinkedIn-style content
  • Customizable absurdity levels from "Mildly Exaggerated" to "Complete Chaos"
  • Topic selection to generate posts about specific professional domains
  • One-click copy functionality for easy sharing
  • Randomized buzzword integration for maximum corporate speak

Technical Details

RottIng was built using a stack of modern technologies that enabled us to create a responsive, powerful application in a short timeframe:

  • Frontend: React.js with Tailwind CSS for the user interface
  • Backend: Node.js with Express for API endpoints
  • Natural Language Processing: Fine-tuned GPT model specialized on LinkedIn content
  • Deployment: Vercel for frontend, Heroku for backend services
  • Version Control: Git and GitHub for collaborative development

Challenges & Solutions

One of the main challenges we faced was training the AI model to generate content that was absurd enough to be funny while still maintaining the recognizable LinkedIn style. We solved this by creating a custom dataset of LinkedIn posts with varying degrees of exaggeration and training our model on this spectrum.

Another challenge was ensuring the generated content wouldn't cross into offensive territory. We implemented content filters and guardrails to keep the humor appropriate while still maintaining the comedic effect.
